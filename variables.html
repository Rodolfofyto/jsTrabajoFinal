La variable

Una variable es un espacio reservado en la memoria del ordenador que nos permite almacenar un valor.

Características

  • El valor almacenado no es fijo y puede variar con el tiempo (de ahí el nombre de variable).
  • Una variable puede ser leída y escrita múltiples veces durante el transcurso del programa.
  • Las variables en JavaScript se declaran antes de ser utilizadas, ya que es en el proceso de declaración cuando se reserva el espacio en memoria.
  • Las variables en cualquier lenguaje de programación disponen de tipo, identificador y valor. Esto es lo que debe ser declarado.
  • En JavaScript, no se indica el tipo del dato, puesto que se determina dinámicamente: var resultado = 0;

Ámbito de variables

  • El ámbito o alcance de una variable es la porción de código en la que existe.
  • Fuera de su ámbito, la variable no podrá ser accedida para lectura ni escritura.
  • Existen por lo general dos ámbitos diferentes en los lenguajes de programación: local y global, aunque también puede existir el ámbito de sentencia de bloque (C, Java…).
    • Ámbito global: en JS la variable tendrá vigencia mientras esté abierta la ventana del navegador.
    • Ámbito local: la variable se encuentra disponible solamente dentro de la función en la que es declarada.
    • Ámbito de sentencia de bloque: la variable existe solamente dentro del bloque en el que ha sido declarada.
  • Tipos de datos en JS
    • String: cadenas de texto.
    • Number: números. En JS todos los números se encuentran en coma flotante de 64 bits de precisión (double). Existen también tres valores adicionales para representar NaN, +Infinity y –Infinity.
    • Boolean: true/false (verdadero/falso).
    • Array: es una lista de elementos de otros tipos.
    • Object: objeto.
    • Null: nulo.
    • Undefined: no definido.
